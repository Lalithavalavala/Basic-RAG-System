# Basic RAG System

## Overview

This project implements a basic Retrieval Augmented Generation (RAG) system using Python, FAISS, and Sentence Transformers.

## Features

- Custom Knowledge Base
- Semantic Search
- FAISS Vector Store
- Question Answering

## Technologies

- Python
- FAISS
- Sentence Transformers
- Google Colab
